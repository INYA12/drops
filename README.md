# 💧 DROPS LAN — Сверхбыстрая передача файлов в локальной сети

**DROPS LAN** — это легкое, невероятно быстрое и красивое веб-приложение для обмена файлами между устройствами, находящимися в одной локальной сети (Wi-Fi, LAN) или виртуальной сети (**Radmin VPN**, Hamachi и др.).

Приложение полностью автономно, не требует установки дополнительных зависимостей и работает на стандартной библиотеке Python 3.

---

## ✨ Основные особенности

*   **Premium интерфейс**: Современный темный дизайн в стиле Glassmorphism с динамическими размытыми сферами на заднем плане, сеткой и плавными микро-анимациями.
*   **Drag & Drop загрузка**: Перетаскивайте файлы прямо в окно браузера для мгновенного добавления в список раздачи.
*   **Потоковое скачивание (Streaming)**: Поддерживает эффективную передачу файлов частями (по 64 КБ), что позволяет передавать файлы любого размера без перегрузки оперативной памяти сервера.
*   **Автоматическое открытие**: При запуске приложение само открывает страницу в вашем браузере по умолчанию.
*   **Поддержка UTF-8 на Windows**: Стартовый скрипт адаптирован для корректного отображения всех русских символов в консоли Windows.
*   **Совместимость с Radmin VPN**: Позволяет обмениваться файлами с друзьями через интернет, находясь в одной виртуальной сети Radmin.

---

## 🛠️ Как запустить

### Быстрый запуск на Windows (рекомендуется)
Просто дважды кликните по файлу:
`start.bat`

Скрипт автоматически настроит кодировку консоли, запустит сервер и откроет его в вашем браузере.

### Запуск через консоль
```bash
python drops.py
```

## 🌐 Как поделиться файлами с другом

### Вариант 1: Внутри домашней сети (один Wi-Fi / роутер)
1. Запустите сервер на своем ПК.
2. Скопируйте ссылку с вашим локальным IP-адресом (например, `http://192.168.0.52:8888`), которая отобразится в консоли или интерфейсе.
3. Отправьте эту ссылку другу. Он сможет зайти и скачать загруженные вами файлы.

### Вариант 2: Через интернет (через Radmin VPN)
1. Вы и ваш друг должны подключиться к одной сети в **Radmin VPN**.
2. Запустите сервер через `start.bat`.
3. Скопируйте ссылку с IP-адресом Radmin VPN (он начинается на **`26.x.x.x`**, например `http://26.7.169.95:8888`).
4. Отправьте ссылку другу.

> ⚠️ **Важно (Брандмауэр Windows)**: При первом запуске брандмауэр Windows может запросить разрешение на доступ к сети для Python. Обязательно разрешите доступ как для **частных**, так и для **публичных** сетей, иначе входящие подключения от друга будут блокироваться.

## 📂 Структура проекта

*   `drops.py` — Самодостаточный исполняемый файл сервера (содержит бэкенд на Python и адаптивный фронтенд внутри).
*   `start.bat` — Скрипт быстрого запуска для Windows с поддержкой UTF-8.
*   `shared_files/` — Локальная папка, куда временно сохраняются загруженные файлы (игнорируется Git).
